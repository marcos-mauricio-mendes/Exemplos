Nessa aula, veremos uma lista de funções para Strings usando PHP. São funções internas que já existem na linguagem. A lista de funções de manipulação de Strings que serão vistas nessa aula é composta pelas instruções:

  • Função strtolower: passa os caracteres para minusculo
  • Função strtoupper: passa os caracteres para maiusculo
  • Função ucfirst: passa a primeira letra para maiusculo
  • Função ucwords: passa as primeiras letras para maiusculo de minha string
  • Função strrev: Passa um string para o contrario
  • Função strpos: Mostra a posição de uma string
  • Função stripos: Mostra a posição de uma string independente do seu tamanho
  • Função substr_count: Conta quantas palavras expecificas tem em minha frase
  • Função substr: Faz o fatiamento de um string dá possição 0 e depois o tamanho escolhido
  • Função str_pad: Faz o tamanho de minha string caber em um espaço selecionado
  • Função str_repeat: Faz a repetição de minha string selecionada
  • Função str_replace: Troca um caractere pre definido de minha string
